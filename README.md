## Sobre o Game Quest Studio
O Game Quest Studio é um projeto focado em ensinar desenvolvimento de jogos de forma prática, acessível e progressiva.

Aqui você encontrará conteúdos sobre programação para jogos, game engines, frameworks e fundamentos utilizados no desenvolvimento de jogos 2D, com foco principalmente em iniciantes e desenvolvedores independentes.

## Aprendizado Prático
O conteúdo do site é desenvolvido através de projetos práticos, séries passo a passo e artigos voltados para conceitos fundamentais do desenvolvimento de jogos.

A proposta é ensinar programação aplicada a jogos de forma clara e direta, permitindo que novos desenvolvedores consigam criar seus próprios projetos enquanto aprendem os fundamentos da área.

## O Que Você Vai Encontrar
Os conteúdos abordam temas como:

- Game loops
- Colisões e hitboxes
- Movimentação de personagens
- Estruturação de projetos
- Game feel
- Organização de código
- Mecânicas para jogos 2D
- Frameworks e engines para desenvolvimento de jogos

## Tecnologias Abordadas
Atualmente, o site possui conteúdos utilizando tecnologias e ferramentas como:

- Python
- Pygame
- Love2D
- Godot
- Defold
- Novas tecnologias e séries práticas poderão ser adicionadas conforme o projeto evolui.

## Objetivo do Projeto
O objetivo do Game Quest Studio é tornar o desenvolvimento de jogos mais acessível, apresentando conceitos de forma progressiva e priorizando o entendimento prático antes de abstrações complexas.
